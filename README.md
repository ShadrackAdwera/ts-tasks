# TS TASKS
Assign tasks to specific users based on their user roles using priority scheduling.

## SERVICES
- Auth
- Section
- Categories
- Tasks
- Scheduler 

**Services communicate asynchronously with each other by emitting events**

## TECHNOLOGIES USED
- Node js
- Express
- Mongo DB
- Typescript
- Redis
- Nats Streaming Server
- Docker
- Kubernetes
